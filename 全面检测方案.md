## 一、后端代码检测

### 1.1 Python 代码检测

#### 语法检测
```bash
python3 -m py_compile <file.py>
```

#### 导入检测
```python
python3 -c "from module import *"
```

#### 单元测试
```bash
pytest tests/ -v
```

## 三、UI 自动化测试

### 3.1 Playwright 测试示例

```python
from playwright.sync_api import sync_playwright

def test_login():
    with sync_playwright() as p:
        browser = p.chromium.launch()
        page = browser.new_page()
        
        # 打开页面
        page.goto("file:///path/to/frontend.html")
        
        # 监听 console
        page.on("console", lambda msg: print(f"{msg.type}: {msg.text}"))
        
        # 点击登录
        page.click("button:has-text('登录')")
        
        # 截图
        page.screenshot(path="login_click.png")
        
        browser.close()
```

### 3.2 检测流程

1. 启动浏览器
2. 打开前端页面
3. 监听 console 错误
4. 模拟用户点击
5. 验证行为正确
6. 截图记录

## 五、问题追踪

### 5.1 问题分类

| 类别 | 严重程度 | 示例 |
|------|----------|------|
| 致命 | 🔴 | 函数未定义、语法错误 |
| 严重 | 🟠 | API路径错误、CORS |
| 一般 | 🟡 | TODO未完成、warnning |
| 提示 | 🔵 | 代码规范建议 |

### 5.2 修复优先级

1. 语法错误 → 无法运行
2. 未定义函数 → 功能失效
3. API错误 → 功能异常
4. 废弃API → 未来兼容
5. 代码规范 → 长期维护
