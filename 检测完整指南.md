## 三、前端检测

### 3.1 HTML结构检测

```python
import re

def check_html_structure(html):
    checks = {
        '<!DOCTYPE': 'DOCTYPE',
        '<html': 'html标签',
        '<head>': 'head标签',
        '<body>': 'body标签',
    }
    for pattern, name in checks.items():
        if pattern.lower() not in html.lower():
            yield f"缺少 {name}"
```

### 3.2 JavaScript函数检测

```python
import re

def check_js_functions(html):
    # 提取定义的函数
    defined = set(re.findall(r'function\s+(\w+)', html))
    async_funcs = set(re.findall(r'async\s+function\s+(\w+)', html))
    all_funcs = defined | async_funcs
    
    # 提取onclick调用
    onclick_calls = re.findall(r'onclick=["\']([^"\']+)["\']', html)
    
    # 检查每个调用是否有定义
    for call in onclick_calls:
        func_name = call.split('(')[0].strip()
        if func_name and func_name not in all_funcs:
            yield f"函数 {func_name}() 未定义"
```

### 3.3 API调用检测

```python
def check_api_calls(html):
    api_patterns = [
        (r'fetch\s*\(', 'fetch'),
        (r'XMLHttpRequest', 'XHR'),
        (r'\$\.ajax', 'jQuery'),
        (r'axios\.', 'axios'),
    ]
    for pattern, name in api_patterns:
        if re.search(pattern, html):
            return True
    return False
```

## 四、UI自动化测试

### 4.1 Playwright基础

```python
from playwright.sync_api import sync_playwright

def test_login():
    with sync_playwright() as p:
        browser = p.chromium.launch()
        page = browser.new_page()
        
        # 打开页面
        page.goto("file:///path/to/frontend.html")
        
        # 监听console
        page.on("console", lambda msg: print(f"{msg.type}: {msg.text}"))
        
        # 监听网络请求
        page.on("request", lambda req: print(f"Request: {req.url}"))
        
        # 点击登录按钮
        page.click("button:has-text('登录')")
        
        # 等待响应
        page.wait_for_timeout(2000)
        
        # 截图
        page.screenshot(path="screenshot.png")
        
        browser.close()
```

### 4.2 检测项目

| 检测项 | 方法 | 预期 |
|--------|------|------|
| 页面加载 | page.goto() | 无错误 |
| Console错误 | page.on("console") | 无error |
| JS异常 | page.on("pageerror") | 无异常 |
| API请求 | page.on("request") | 正确URL |
| 按钮点击 | page.click() | 有响应 |
| 表单提交 | page.fill() + submit | 数据正确 |
