## 项目简介

VisionClaw是基于OpenClaw架构的视觉智能体移动端平台，是OpenClaw在物理世界的延伸。

## 核心能力

- 视觉感知：摄像头、视频流、图像分析
- 语音交互：语音识别、语音合成
- 设备控制：拍照、录像、传感器
- 云端推理：LLM对话、多模态理解

## 技术架构

- 前端：Flutter + Kotlin
- 后端：Python FastAPI
- 存储：PostgreSQL + Redis + Milvus
- AI：YOLO + Whisper + LLaVA + DeepSeek
