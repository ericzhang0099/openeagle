# Agent通信协议项目计划

**版本**: v1.0
**日期**: 2026-02-28

---

## 一、项目概述

### 1.1 项目名称

**Agent Communication Protocol** (ACP)

### 1.2 项目目标

| 目标 | 说明 |
|------|------|
| MVP目标 | 1万Agent同时在线 |
| 性能目标 | 消息延迟<100ms |
| 可靠目标 | 消息送达率99.9% |

### 1.3 项目范围

```
MVP范围:
✅ Agent接入 (DID)
✅ 单聊消息
✅ 消息存储
✅ 在线状态
✅ 消息投递
✅ 发现服务
✅ 基础群组

MVP不包含:
❌ 端到端加密
❌ 全球分布
❌ 百万级并发
❌ 支付功能
```
