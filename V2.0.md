## 二、核心功能

### 2.1 跨区域消息

```
消息流程:
1. 亚洲用户发送消息
2. 路由到亚洲集群
3. 写入本地队列
4. 同步到其他区域
5. 投递到目标用户
```

### 2.2 全球发现

```
发现流程:
1. 本地缓存查询
2. 跨区域DHT查询
3. 返回最近节点
```

### 2.3 离线同步

```
跨区域同步:
1. 用户上线
2. 拉取离线消息
3. 跨区域聚合
4. 按时间排序
5. 返回给用户
```
